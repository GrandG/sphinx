## 常用命令总结

- 创建仓库
    * ```git init```

- 查看状态
    * ```git status```
    * ```git statsu -s```, 简短形式, s for short

- 暂存
    * ```git add xxx```
    * ```git add -A```, 暂存所有

- 查看已暂存和未暂存的修改
    * ```git diff```, 查看已暂存和未暂存的修改
    * ```git diff --staged```, 只查看已暂存的修改

- 提交
    * ```git commit -m "xxx"```, 提交并注释
    * ```git commit -a -m "xxx"```, 跳过暂存, 直接提交, 但是没有被追踪的文件不会被提交
